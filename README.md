# TCC Ipet Mobile

- React Native <3
- Styled Components

# Requerimento/Dependências

Antes de começar a rodar a aplicação, é muito importante que você tenha a versão mais atualizada do node em seu computador.

# Rodando o Projeto
Você ira ter instalado:
expo e yarn

Para rodar: 
`yarn start` ou `expo start`
